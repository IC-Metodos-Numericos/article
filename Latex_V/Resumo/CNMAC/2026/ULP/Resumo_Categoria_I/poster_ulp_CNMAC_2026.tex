Este trabalho apresenta resultados de estudos realizados em um projeto de iniciação científica cujo objetivo é explorar graficamente alguns efeitos do cálculos realizados com ponto flutuante e das Unidades de Última Posição (Units of Last Place - ULP).

Um sistema de ponto flutuante é a maneira utilizada pelos computadores para representar números reais através de uma notação compacta e eficaz, padronizado pelo IEEE-754. Ele permite escrever números de grandezas diversas utilizando apenas números inteiros. No entanto, essa representação fica limitada ao adequar o número ao sistema adotado, causando erros que podem se acumular em operações sucessivas, produzindo resultados imprecisos ou incorretos.

Um sistema de ponto flutuante pode ser definido da seguinte forma:
\begin{equation}
\mathbb{F}(\beta, t, L, U) = \{0\} \cup \{\pm d_1.d_2d_3 \ldots d_t \times \beta^e : d_1 \neq 0, d_i \in \{0, 1, \ldots, \beta-1\}, e = L, L+1, \ldots, U\},
\end{equation}
onde $\beta$ é a base, $t$ é o número de dígitos significativos, $L$ é o menor expoente e $U$ é o maior expoente. 

O erro de ponto flutuante pode ser definido como a diferença entre o valor real e o valor representado em ponto flutuante. 

\begin{equation}
Erro = \left|\bar{x} - x\right|,
\end{equation}

A ULP é a menor diferença entre dois números representáveis em ponto flutuante, ou seja, a distância entre um número e o próximo número representável. Pela natureza discreta do sistema de ponto flutuante, a ULP é uma medida importante para avaliar a precisão dos cálculos realizados com ponto flutuante. A ULP pode ser definida como:
\begin{equation}
ULP(x) = \beta^{e - t + 1},
\end{equation}

Vamos analisar o número $5.5^{23}$.

O número $5.5^{23}$ é um número muito grande, e quando representado em ponto flutuante, ele pode ser aproximado para um valor próximo a $1.0 \times 10^{16}$.

A ULP para o número $5.5^{23}$ pode ser calculada como:
\begin{equation}
ULP(5.5^{23}) = \beta^{e - t + 1},
\end{equation}



onde $e$ é o expoente do número $5.5^{23}$ e $t$ é o número de dígitos significativos.
